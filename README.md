# scrumlr2pdf

This tool provides a convenient way to transform Markdown exports from the
awesome online retrospective tool [scrumlr.io](https://scrumlr.io/) into a
neatly formatted PDF file. It uses pandoc running in a Docker container to
ensure a consistent environment for conversion. Ideal for archiving or sharing
retrospective results easily.

> [!IMPORTANT]
> This is an unofficial, third-party tool not affiliated with, endorsed by, or
> sponsored by Scrumlr or its creators.

## Usage

1. Enjoy your Retrospective.
1. Export your Scrumlr board as markdown.
1. Place the `.md` file in this directory.
1. `./run.sh`
1. Profit.
